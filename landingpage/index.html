Hera

A self-improving, multi-platform AI agent built in Go

Get Started Documentation

Features

15+ LLM Providers

OpenAI, Anthropic, Gemini, Mistral, OpenRouter, Ollama, and more. Switch models with a single config change.

18 Platforms

CLI, Telegram, Discord, Slack, WhatsApp, Signal, Matrix, Email, SMS, and more. One agent, everywhere.

40+ Tools

File operations, web search, code execution, image generation, voice synthesis, and more built-in.

Persistent Memory

SQLite with FTS5 full-text search. Remember conversations, facts, and context across sessions.

170+ Skills

DevOps, ML/AI, research, creative, productivity, and more. Plus agent-created skills that grow over time.

Single Binary

Built in Go for performance. No runtime dependencies. Deploy anywhere with a single binary or Docker.

Quick Start

$ go install github.com/sadewadee/hera/cmd/hera@latest
$ export OPENAI_API_KEY=sk-...
$ hera chat
> Hello! I'm Hera. How can I help you today?
