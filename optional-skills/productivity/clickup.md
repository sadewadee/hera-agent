# ClickUp workspace management

## Purpose
ClickUp workspace management for team and personal productivity through project management tools.

## Instructions
1. Connect to the platform API
2. Query and manage tasks, projects, and workflows
3. Create and update items based on user requests
4. Sync status across tools and platforms
5. Generate reports and insights

## Task Management
- Create tasks with proper metadata (priority, assignee, due date)
- Organize tasks into projects, sprints, or boards
- Track progress and update statuses
- Set up automations for recurring workflows
- Link related tasks and dependencies

## Reporting
- Sprint velocity and burndown charts
- Task completion rates and trends
- Blocked items and bottleneck identification
- Team workload distribution
- Overdue items and deadline tracking

## Integrations
- GitHub/GitLab for code-linked tasks
- Slack/Teams for notifications
- Calendar for deadline management
- CI/CD for deployment tracking
- Time tracking tools

## Best Practices
- Keep task descriptions clear and actionable
- Update status regularly to maintain visibility
- Use labels and tags for filtering
- Review and close stale items regularly
- Automate status transitions where possible
