# YouTube RSS Feeds

## Purpose
Subscribe to YouTube channels via RSS feeds for notification-free, algorithm-free content consumption.

## Instructions
1. Convert YouTube channel URLs to RSS feed URLs
2. Add feeds to your preferred RSS reader
3. Filter videos by title keywords or duration
4. Organize channels into categories
5. Set up notifications for priority channels

## RSS URL Format
```
https://www.youtube.com/feeds/videos.xml?channel_id=CHANNEL_ID
```

## Best Practices
- Group channels by topic for organized reading
- Use keyword filters to surface relevant videos
- Check feeds regularly rather than relying on notifications
- Export feed list as OPML for backup
