# Firewall configuration and management

## Purpose
Firewall configuration and management for system administration and security.

## Instructions
1. Identify the system task or configuration need
2. Review current settings and configuration
3. Apply changes following best practices
4. Test and verify the configuration
5. Document the changes

## Common Operations
- Configuration review and setup
- Security hardening
- Troubleshooting connectivity issues
- Monitoring and alerting
- Backup and recovery

## Security
- Follow principle of least privilege
- Use key-based authentication where possible
- Monitor and audit access logs
- Keep systems patched and updated
- Document all security configurations

## Best Practices
- Test configuration changes in non-production first
- Keep documentation current
- Automate routine tasks
- Monitor for anomalies
- Have rollback plans for changes
