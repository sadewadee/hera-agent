# Smart Thermostat Control

## Purpose
Control and automate smart thermostats for comfort and energy efficiency.

## Instructions
1. Query current temperature and thermostat settings
2. Adjust temperature based on user request
3. Create or modify schedules
4. Set up automation rules based on occupancy or time
5. Monitor energy usage patterns

## Features
- Temperature adjustment (heat, cool, auto modes)
- Schedule creation and modification
- Away mode and geofencing
- Energy usage reporting
- Integration with other smart home devices

## Best Practices
- Use schedules rather than manual adjustments
- Set reasonable temperature ranges for efficiency
- Enable learning features if available
- Monitor energy usage trends
- Integrate with occupancy sensors for optimal comfort
