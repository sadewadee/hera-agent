# Event-driven architecture patterns

## Purpose
Event-driven architecture patterns for building robust, scalable software systems.

## Instructions
1. Assess the architectural requirements
2. Design following established patterns
3. Implement with proper testing and error handling
4. Review for security and performance

## Architecture

- Implement proper monitoring
