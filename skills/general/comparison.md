# Comparison and decision analysis

## Purpose
Comparison and decision analysis as a general-purpose assistant capability.

## Instructions
1. Understand the user's specific request and context
2. Apply the appropriate technique for the task
3. Provide clear, well-structured output
4. Offer follow-up suggestions for refinement
5. Adapt to the user's preferred format and level of detail

## Approach
- Start by clarifying ambiguous requests
- Use structured formats (lists, tables, headers) for clarity
- Provide examples when helpful
- Offer multiple options when appropriate
- Explain reasoning when it adds value

## Quality Standards
- Accuracy and factual correctness
- Clear and concise communication
- Appropriate level of detail for the audience
- Consistent formatting and structure
- Actionable and practical output

## Best Practices
- Confirm understanding before proceeding with complex tasks
- Break large tasks into manageable steps
- Provide context for recommendations
- Flag uncertainties and limitations
- Suggest resources for further exploration
