# Graph and network analysis

## Purpose
Apply graph analysis methods for data-driven insights and decision making.

## Instructions
1. Define the analytical objective
2. Prepare and validate the data
3. Apply appropriate methods
4. Validate results with statistical rigor
5. Communicate findings clearly

## Methodology
- Define clear success criteria
- Use cross-validation for evaluation
- Compare against baselines
- Report uncertainty and limitations
- Document all steps for reproducibility

## Best Practices
- Start with exploratory analysis
- Validate assumptions before applying methods
- Use multiple approaches for robustness
- Report results honestly including failures
- Make analysis reproducible
