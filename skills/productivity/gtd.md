# Getting Things Done methodology

## Purpose
Getting Things Done methodology for improved personal and team effectiveness.

## Instructions
1. Assess current productivity challenges and goals
2. Recommend appropriate frameworks and techniques
3. Help implement the chosen methodology
4. Track progress and adjust approach
5. Build sustainable habits and systems

## Methodology
- Define clear objectives and success criteria
- Break large goals into actionable tasks
- Prioritize based on impact and urgency
- Schedule focused work blocks
- Review and retrospect regularly

## Time Management
- Identify and protect peak productivity hours
- Batch similar tasks to minimize context switching
- Set realistic deadlines with buffer time
- Use timeboxing for open-ended tasks
- Regular breaks for sustained performance

## Tools and Systems
- Task management apps and workflows
- Calendar blocking and scheduling
- Automation for repetitive tasks
- Templates for common workflows
- Metrics and dashboards for tracking

## Best Practices
- Start with one system and master it
- Review and adjust weekly
- Minimize work-in-progress
- Protect deep work time
- Document processes for consistency
