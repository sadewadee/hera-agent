# Speech-to-text recognition

## Purpose
Speech-to-text recognition for multimedia content creation and processing.

## Instructions
1. Identify the media type and desired output format
2. Select the appropriate processing tool or API
3. Configure parameters for quality and format
4. Process the media with proper error handling
5. Deliver the output in the requested format

## Supported Formats
- Images: PNG, JPEG, WebP, SVG, GIF
- Audio: MP3, WAV, FLAC, OGG, AAC
- Video: MP4, WebM, MOV, AVI
- Documents: PDF (via OCR/conversion)

## Processing Capabilities
- Format conversion and transcoding
- Quality adjustment and optimization
- Metadata extraction and modification
- Batch processing for multiple files
- Cloud API integration for advanced features

## Quality Considerations
- Balance quality vs file size for intended use
- Preserve metadata when converting formats
- Use lossless formats for archival purposes
- Optimize for web delivery when appropriate
- Test output across target platforms

## Best Practices
- Validate input file integrity before processing
- Provide progress feedback for long operations
- Handle large files with streaming where possible
- Respect copyright and licensing of source material
- Clean up temporary files after processing
