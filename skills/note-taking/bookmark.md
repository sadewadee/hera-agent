# Bookmark Management

## Purpose
Organize and manage web bookmarks for efficient information retrieval.

## Instructions
1. Capture bookmarks with metadata
2. Categorize and tag for easy retrieval
3. Periodically review and prune
4. Search across bookmark collection
5. Export and sync across devices

## Organization
- Use folders for broad categories
- Tag for cross-cutting topics
- Add descriptions for context
- Archive inactive bookmarks
- Remove broken links regularly

## Best Practices
- Process the inbox regularly
- Use consistent naming conventions
- Include context when saving
- Review and prune quarterly
- Back up bookmark collection
