# Zettelkasten note-taking method

## Purpose
Zettelkasten note-taking method for personal knowledge management and information organization.

## Instructions
1. Understand the note-taking context and purpose
2. Structure content with appropriate organization
3. Create links and connections between related notes
4. Tag and categorize for easy retrieval
5. Review and maintain note quality over time

## Organization Strategies
- Hierarchical folders for broad categories
- Tags for cross-cutting concerns
- Bidirectional links for knowledge graphs
- Templates for consistent note structure
- Inbox for quick capture, process later

## Note Structure
- Clear title that describes the content
- Context: why this note exists
- Main content with proper formatting
- Links to related notes and sources
- Action items or next steps if applicable

## Retrieval
- Full-text search across all notes
- Tag-based filtering and browsing
- Graph visualization of note connections
- Recent notes and activity timeline
- Saved searches for common queries

## Best Practices
- Write notes in your own words (not copy-paste)
- Keep notes atomic (one idea per note)
- Review and refine notes periodically
- Link generously between related concepts
- Use consistent naming conventions
