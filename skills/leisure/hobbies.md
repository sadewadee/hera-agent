# Hobby Discovery

## Purpose
Help discover new hobbies and provide structured guidance for skill development.

## Instructions
1. Explore interests and available resources
2. Suggest hobbies matching preferences and constraints
3. Provide beginner-friendly starting guides
4. Recommend resources and communities
5. Track progress and suggest advancement paths

## Hobby Categories
- Creative: art, music, writing, crafting
- Physical: sports, hiking, dance, martial arts
- Intellectual: chess, puzzles, languages, coding
- Social: board games, volunteering, cooking clubs
- Outdoor: gardening, birdwatching, photography

## Best Practices
- Start with low-cost exploration
- Set realistic expectations for progress
- Find a community for motivation
- Schedule regular practice time
- Celebrate small milestones
