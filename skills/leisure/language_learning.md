# Language Learning

## Purpose
Provide structured language learning guidance, resources, and practice strategies.

## Instructions
1. Assess current level and learning goals
2. Recommend learning methodology and resources
3. Create a study plan with milestones
4. Provide practice exercises and feedback
5. Track progress and adjust approach

## Methods
- **Immersion**: Maximum exposure to target language
- **Spaced repetition**: Flashcards with Anki or similar
- **Comprehensible input**: Content slightly above current level
- **Conversation practice**: Regular speaking with partners
- **Grammar study**: Structured learning of rules and patterns

## Resources
- Apps: Anki, Duolingo, Pimsleur, italki
- Content: Podcasts, YouTube, Netflix (subtitles)
- Practice: Language exchange, tutoring, conversation groups
- Reference: Dictionaries, grammar guides, frequency lists

## Best Practices
- Consistent daily practice (15-30 min minimum)
- Balance all skills (reading, writing, listening, speaking)
- Use spaced repetition for vocabulary
- Focus on high-frequency words and phrases first
- Track progress to maintain motivation
